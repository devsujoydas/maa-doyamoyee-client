## 📄 Project Overview

Maa-Doyamoyee is a **modern temple website** designed to provide information about the temple, manage community members, and handle blogs, notices, donations, and events.  
It supports **multiple languages** using **i18n** and offers a **full-featured admin panel** for management.

---

## 🧩 Features

- 🌐 Multi-language support with **i18next**  
- 📱 Fully responsive **Tailwind CSS** design  
- 🧭 Smooth navigation with **React Router DOM**  
- ⚡ Client-side state management with **React Query**  
- 📝 Forms & validation with **React Hook Form**  
- 🎨 Interactive UI with **Framer Motion**, **Swiper**, **Lucide Icons**, **React Icons**, **Fancybox**  
- 🔔 Real-time notifications using **React Hot Toast**  
- 🛡️ Secure admin panel with structured management  
- 🖼️ Media gallery for **images & videos**  
- 💰 Donation handling (via external integration)  
- 👤 User authentication: **SignUp**, **SignIn**, **ForgotPassword**  
- 🏷️ Profile page for individual users  
- 📰 **Blogs** with details pages  
- 📢 **Notice Details** page  
- 📆 **Events Details** page  

---

## 🏗️ Tech Stack

- **⚛️ Frontend:** React 18 + JSX  
- **🛣️ Routing:** React Router DOM v7  
- **📡 State & API:** Axios, React Query  
- **🎨 Styling:** Tailwind CSS v4, Flowbite React components  
- **📝 Forms & Validation:** React Hook Form  
- **🎞️ Animations & UI:** Framer Motion, Swiper, Lucide React, React Icons  
- **🌐 i18n:** i18next, react-i18next, i18next-browser-languagedetector  
- **🔔 Notifications:** React Hot Toast  
- **🛠️ Build Tool:** Vite

## ⚡ Installation

```bash
# Clone the repository
git clone https://github.com/devsujoydas/maa-doyamoyee-client.git
cd maa-doyamoyee-client

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
